● Nível Simples O que é uma classe?

Uma classe é um molde para criar objetos. Define os dados que cada objeto terá (atributos) e as ações que ele pode fazer (métodos). É o fundamento da Programação Orientada a Objetos (POO).

nivel-simples.ts 
class Pessoa {
  // Atributos
  nome: string
  idade: number

  // Construtor: executado ao criar o objeto
  constructor(nome: string, idade: number) {
    this.nome  = nome
    this.idade = idade
  }

  // Método
  seApresentar(): void {
    console.log(`Sou ${this.nome}, ${this.idade} anos.`)
  }
}

// Criar objetos (instâncias)
const pessoaUm = new Pessoa("Alice", 25)
const pessoaDois = new Pessoa("Bruno", 30)

// Acessar propriedades
console.log(pessoaUm.nome)   // "Alice"
pessoaDois.seApresentar()    // "Sou Bruno, 30 anos."
● Nível Médio Herança, private e protected

A herança permite criar uma nova classe baseada em outra. Os modificadores private, protected e public controlam quem pode acessar os atributos e métodos.

nivel-medio.ts 
class ContaBancaria {
  protected titular: string
  private   saldo: number       // só acessível aqui dentro

  constructor(titular: string, deposito: number) {
    this.titular = titular
    this.saldo   = deposito
  }

  depositar(valor: number): void {
    if (valor > 0) this.saldo += valor
  }

  sacar(valor: number): boolean {
    if (valor > this.saldo) return false
    this.saldo -= valor
    return true
  }

  obterSaldo(): number { return this.saldo }
}

// Herança: ContaCorrente estende ContaBancaria
class ContaCorrente extends ContaBancaria {
  private limiteCredito: number

  constructor(titular: string, deposito: number, limite: number) {
    super(titular, deposito)   // chama o construtor pai
    this.limiteCredito = limite
  }

  sacarComLimite(valor: number): boolean {
    const totalDisponivel = this.obterSaldo() + this.limiteCredito
    if (valor > totalDisponivel) return false
    return super.sacar(valor)
  }
}
● Nível Avançado Classes abstratas e interfaces

Uma classe abstrata define uma estrutura que subclasses devem implementar. Uma interface define um contrato de forma que qualquer objeto que a implemente garante ter certos métodos e propriedades.

nivel-avancado.ts 
// Interface: define um contrato
interface FormaGeometrica {
  calcularArea(): number
  calcularPerimetro(): number
  descricao: string
}

// Classe abstrata: base com método abstrato
abstract class Forma implements FormaGeometrica {
  abstract calcularArea(): number
  abstract calcularPerimetro(): number

  get descricao(): string {
    return `Área: ${this.calcularArea().toFixed(2)}`
  }
}

// Implementações concretas
class Circulo extends Forma {
  constructor(private raio: number) { super() }
  calcularArea() { return Math.PI * this.raio ** 2 }
  calcularPerimetro() { return 2 * Math.PI * this.raio }
}

class Retangulo extends Forma {
  constructor(private largura: number, private altura: number) { super() }
  calcularArea() { return this.largura * this.altura }
  calcularPerimetro() { return 2 * (this.largura + this.altura) }
}

// Polimorfismo: trata os dois como FormaGeometrica
const listaFormas: FormaGeometrica[] = [new Circulo(5), new Retangulo(4, 6)]
listaFormas.forEach(forma => console.log(forma.calcularArea()))
// 78.54, 24

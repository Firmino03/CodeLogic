● Nível Simples Métodos essenciais de string

Strings são textos. TypeScript herda todos os métodos de string do JavaScript, permitindo buscar, transformar e formatar textos com facilidade.

nivel-simples.ts 
const frase = "  Olá, Mundo TypeScript!  "

frase.length                       // 26
frase.trim()                        // "Olá, Mundo TypeScript!"
frase.toUpperCase()                // "  OLÁ, MUNDO TYPESCRIPT!  "
frase.toLowerCase()                // "  olá, mundo typescript!  "
frase.includes("Mundo")            // true
frase.replace("Mundo", "Brasil")   // "  Olá, Brasil TypeScript!  "

// Template string: inserir variável no texto
const nomeJogador = "Carlos"
const pontos       = 100
const mensagem     = `${nomeJogador} fez ${pontos} pontos!`
● Nível Médio split, slice, padStart e mais
nivel-medio.ts 
// Dividir em array e juntar de volta
const csvTexto = "Ana,Bruno,Carla,Diego"
const listaDeNomes = csvTexto.split(",")
// ["Ana", "Bruno", "Carla", "Diego"]

const textoReunido = listaDeNomes.join(" | ")
// "Ana | Bruno | Carla | Diego"

// Extrair parte da string
const email = "usuario@dominio.com"
const usuario = email.slice(0, email.indexOf("@"))  // "usuario"
const dominio  = email.slice(email.indexOf("@") + 1)    // "dominio.com"

// Preencher para formatação
"7".padStart(3, "0")   // "007" — útil para IDs e datas
"5".padEnd(5, ".")     // "5...."

// Verificar início e fim
email.startsWith("usuario")    // true
email.endsWith(".com")         // true
● Nível Avançado Expressões regulares (RegEx)

Expressões regulares são padrões para buscar e transformar texto. São muito poderosas para validação de dados (email, CPF, telefone) e extração de informações.

nivel-avancado.ts 
// Validar formato de email
function emailEValido(email: string): boolean {
  const padraoEmail = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
  return padraoEmail.test(email)
}
emailEValido("ana@email.com")   // true
emailEValido("invalido")       // false

// Extrair todos os números de um texto
const textoComNúmeros = "Pedido #1234 com 3 itens, total R$ 99.90"
const numerosEncontrados = textoComNúmeros.match(/\d+(\.\d+)?/g)
// ["1234", "3", "99.90"]

// Substituir padrão com função
const textoFormatado = "nome_do_campo".replace(/_([a-z])/g, (_, letra) => letra.toUpperCase())
// "nomeDoCampo" (snake_case → camelCase)

// Grupos nomeados para extração precisa
const padraoData = /(?<dia>\d{2})\/(?<mes>\d{2})\/(?<ano>\d{4})/
const resultadoData = "25/12/2024".match(padraoData)
console.log(resultadoData?.groups)
// { dia: "25", mes: "12", ano: "2024" }

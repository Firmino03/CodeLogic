● Nível Simples if, else e else if

Condicionais permitem ao programa tomar decisões: executar um bloco apenas se uma condição for verdadeira. São a base do fluxo de controle de qualquer programa.

nivel-simples.ts 
const idadeUsuario: number = 16

// if / else
if (idadeUsuario >= 18) {
  console.log("Acesso permitido")
} else {
  console.log("Acesso negado — menor de idade")
}

// else if: múltiplas condições
const notaFinal: number = 75

if (notaFinal >= 90) {
  console.log("Conceito A")
} else if (notaFinal >= 80) {
  console.log("Conceito B")
} else if (notaFinal >= 70) {
  console.log("Conceito C")   // ← este será executado
} else {
  console.log("Reprovado")
}
● Nível Médio switch/case e ternário

O switch é mais legível que vários else if quando você está comparando um único valor contra múltiplas opções. O ternário é um if/else compacto para uma linha.

nivel-medio.ts 
// switch / case
const diaDaSemana: number = 3
let nomesDoDia: string

switch (diaDaSemana) {
  case 1: nomesDoDia = "Segunda"; break
  case 2: nomesDoDia = "Terça";   break
  case 3: nomesDoDia = "Quarta";  break
  case 4: nomesDoDia = "Quinta";  break
  case 5: nomesDoDia = "Sexta";   break
  default: nomesDoDia = "Fim de semana"
}

// Ternário: expressão compacta
const temperatura = 32
const sensacaoTermica = temperatura > 30
  ? "Calor intenso"
  : temperatura > 20
    ? "Agradável"
    : "Frio"
● Nível Avançado Type narrowing e discriminated unions

O TypeScript usa o resultado de condicionais para estreitar (narrow) o tipo de uma variável. Discriminated unions é um padrão onde um campo em comum define qual tipo está sendo usado.

nivel-avancado.ts 
// Type narrowing: TypeScript sabe o tipo dentro do if
function processarValor(entrada: string | number): string {
  if (typeof entrada === "string") {
    return entrada.toUpperCase()  // TypeScript sabe: é string aqui
  }
  return entrada.toFixed(2)      // TypeScript sabe: é number aqui
}

// Discriminated union: campo "tipo" determina a forma
type EventoSistema =
  | { tipo: "login";   nomeUsuario: string }
  | { tipo: "compra";  valorTotal: number }
  | { tipo: "erro";    mensagemErro: string }

function registrarEvento(evento: EventoSistema): void {
  switch (evento.tipo) {
    case "login":
      console.log(`Login: ${evento.nomeUsuario}`)  // TS sabe que tem nomeUsuario
      break
    case "compra":
      console.log(`Compra: R$ ${evento.valorTotal}`)   // TS sabe que tem valorTotal
      break
    case "erro":
      console.error(evento.mensagemErro)
      break
  }
}

● Nível Simples Os quatro grupos de operadores

Operadores são símbolos que realizam operações sobre valores. Os mais usados no dia a dia são os aritméticos, de atribuição, relacionais e lógicos.

Aritméticos
  • +soma
  • -subtração
  • *multiplicação
  • /divisão
  • %resto da divisão
Atribuição
  • =atribuir valor
  • +=somar e atribuir
  • -=subtrair e atribuir
  • *=multiplicar e atribuir
Relacionais
  • ===igual (valor e tipo)
  • !==diferente
  • >maior que
  • <menor que
  • >=maior ou igual
  • <=menor ou igual
Lógicos
  • &&E — AND
  • ||OU — OR
  • !NÃO — NOT
nivel-simples.ts 
const soma        = 10 + 3    // 13
const restoDivisao = 10 % 3    // 1

let pontuacao = 0
pontuacao += 10              // 10
pontuacao *= 2               // 20

5 === 5    // true
5 !== 3    // true

const temIdade  = 20 >= 18           // true
const temRenda  = 3000 > 1500        // true
const aprovado  = temIdade && temRenda // true
⚠️

Sempre use === (triplo igual) no TypeScript — ele compara valor e tipo, evitando surpresas como 0 == false ser true.

● Nível Médio Operador ternário e nullish

O operador ternário é um if/else compactado em uma linha. O nullish coalescing (??) retorna o lado direito apenas se o lado esquerdo for null ou undefined.

nivel-medio.ts 
// Ternário: condição ? se_verdadeiro : se_falso
const notaAluno = 75
const situacao  = notaAluno >= 60 ? "Aprovado" : "Reprovado"

// Nullish coalescing: usa valor padrão se null/undefined
const nomeSalvo: string | null = null
const nomeExibido = nomeSalvo ?? "Visitante"  // "Visitante"

// Optional chaining: acessa propriedade só se existir
const usuario: { endereco?: { cidade: string } } = {}
const cidadeUsuario = usuario.endereco?.cidade  // undefined (sem erro)

// Encadeamento: ?? com ?.
const cidadeOuPadrao = usuario.endereco?.cidade ?? "Não informada"
● Nível Avançado Operadores bit a bit e de tipo

Os operadores bit a bit manipulam os bits individuais de números. São usados em otimizações de performance, permissões em sistemas e manipulação de flags. O operador typeof e instanceof ajudam a verificar tipos em tempo de execução.

nivel-avancado.ts 
// Sistema de permissões com flags binárias
const PERMISSAO_LER     = 0b001  // 1
const PERMISSAO_ESCREVER = 0b010  // 2
const PERMISSAO_EXECUTAR = 0b100  // 4

// Combinar permissões com OR bit a bit
const permissoesAdmin = PERMISSAO_LER | PERMISSAO_ESCREVER | PERMISSAO_EXECUTAR
// 7 = 0b111

// Verificar permissão com AND bit a bit
const podeLer     = (permissoesAdmin & PERMISSAO_LER) !== 0     // true
const podeExecutar = (permissoesAdmin & PERMISSAO_EXECUTAR) !== 0 // true

// typeof — verifica tipo em tempo de execução
function processarEntrada(valor: string | number): string {
  if (typeof valor === "string") {
    return valor.toUpperCase()
  }
  return valor.toFixed(2)
}

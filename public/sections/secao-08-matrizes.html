● Nível Simples O que é uma matriz?

Uma matriz é um array de arrays — uma tabela com linhas e colunas. Ideal para representar grades, tabuleiros de jogos ou tabelas de dados.

nivel-simples.ts 
// Matriz 3x3 — 3 linhas, 3 colunas
const tabelaNumeros: number[][] = [
  [1, 2, 3],   // linha 0
  [4, 5, 6],   // linha 1
  [7, 8, 9]    // linha 2
]

// Acesso: [linha][coluna]
console.log(tabelaNumeros[0][0])  // 1
console.log(tabelaNumeros[1][2])  // 6
console.log(tabelaNumeros[2][1])  // 8

// Percorrer todos os elementos
for (const linhaAtual of tabelaNumeros) {
  for (const valorCelula of linhaAtual) {
    process.stdout.write(valorCelula + " ")
  }
  console.log()
}
// 1 2 3
// 4 5 6
// 7 8 9
● Nível Médio Tabuleiro de jogo e operações

Matrizes são perfeitas para representar jogos como jogo da velha, xadrez ou qualquer grade. Podemos verificar linhas, colunas e diagonais iterando pelos índices.

nivel-medio.ts 
type Celula = "X" | "O" | null
const tabuleiroJogo: Celula[][] = [
  ["X", "O", "X"],
  ["O", "X", null],
  [null, "O", "X"]
]

// Verificar diagonal principal (canto sup-esq → inf-dir)
function verificarDiagonal(tabuleiro: Celula[][]): Celula | null {
  const tamanho = tabuleiro.length
  const primeiraCelula = tabuleiro[0][0]

  for (let i = 1; i < tamanho; i++) {
    if (tabuleiro[i][i] !== primeiraCelula) return null
  }
  return primeiraCelula   // vencedor!
}

// Somar todos os valores de uma matriz
function somarMatriz(matriz: number[][]): number {
  return matriz.flat().reduce((soma, valor) => soma + valor, 0)
}
● Nível Avançado Multiplicação de matrizes

A multiplicação de matrizes é fundamental em computação gráfica, machine learning e processamento de sinais. Para multiplicar A × B, o número de colunas de A deve ser igual ao número de linhas de B.

nivel-avancado.ts 
function multiplicarMatrizes(
  matrizA: number[][],
  matrizB: number[][]
): number[][] {
  const linhasA = matrizA.length
  const colunasA = matrizA[0].length
  const colunasB = matrizB[0].length

  // Criar resultado zerado
  const resultado: number[][] =
    Array.from({ length: linhasA }, () => new Array(colunasB).fill(0))

  for (let linha = 0; linha < linhasA; linha++) {
    for (let coluna = 0; coluna < colunasB; coluna++) {
      for (let k = 0; k < colunasA; k++) {
        resultado[linha][coluna] += matrizA[linha][k] * matrizB[k][coluna]
      }
    }
  }
  return resultado
}

const A = [[1, 2], [3, 4]]
const B = [[5, 6], [7, 8]]
multiplicarMatrizes(A, B)
// [[19, 22], [43, 50]]

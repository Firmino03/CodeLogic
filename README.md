## CodeLogic — Reforço de Fundamentos em Lógica com TypeScript

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)

> Projeto acadêmico desenvolvido com o objetivo de reforçar e consolidar os conteúdos aprendidos na disciplina de Lógica de Programação, utilizando exclusivamente TypeScript.


## Sobre o Projeto

O **CodeLogic** foi criado como uma iniciativa de reforço acadêmico com foco na consolidação dos conceitos ensinados em sala de aula e no material disponibilizado pelo professor.

A proposta do projeto é revisitar os fundamentos da lógica de programação por meio de exemplos práticos, organizados de forma didática, utilizando a mesma linguagem trabalhada na disciplina: **TypeScript**.

Este projeto tem como principal objetivo fortalecer a base lógica por meio da prática estruturada, organização de ideias e aplicação direta dos conceitos estudados.


##  Objetivo Acadêmico

- Reforçar os conteúdos abordados em sala.
- Aplicar na prática os conceitos presentes no material de apoio.
- Consolidar o raciocínio lógico por meio da implementação.
- Demonstrar comprometimento com o processo de aprendizagem.
- Transformar teoria em prática de forma organizada e progressiva.

## Como Rodar

### Pré-requisitos
- **Node.js** instalado (versão 18 ou superior)
- **npm** (já vem com o Node.js)

### Passos

```bash
# 1. Entre na pasta do projeto
cd projeto-logica

# 2. Instale as dependências
npm install

# 3. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse **http://localhost:5173** no navegador.
